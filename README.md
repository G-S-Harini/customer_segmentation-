# Customer Segmentation Dashboard

##  Project Overview
This project focuses on customer segmentation and behavior analysis using the Mall Customers dataset. The dashboard was developed in Power BI to analyze customer demographics, income levels, spending patterns, and customer distribution.

##  Objective
To segment customers based on their demographics and purchasing behavior and derive meaningful business insights through data visualization.

##  Tools Used
- Power BI
- Microsoft Excel
- Python (Basic Data Analysis)

##  Dataset
- Mall Customers Dataset

##  Dashboard Features

### KPI Cards
- Total Customers
- Average Annual Income
- Average Spending Score

### Visualizations
- Income vs Spending Score (Scatter Chart)
- Customer Distribution (Donut Chart)
- Customer Count Analysis (Bar Chart)
- Customer Age Distribution (Treemap)

## Key Insights
- Customer spending behavior varies across different income levels.
- Age groups show different customer distribution patterns.
- Customer demographics provide useful insights for targeted marketing.
- Income and spending analysis help identify potential customer segments.

##  Outcome
Successfully built an interactive Customer Segmentation Dashboard that helps understand customer behavior, demographics, and spending patterns for data-driven decision making.
